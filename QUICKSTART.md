# ⚡ Quick Setup Guide\n\nGet Turf Analytics running in minutes!\n\n## 🚀 Prerequisites\n\n- **Python 3.9+** - [Download](https://www.python.org/downloads/)\n- **Node.js 16+** - [Download](https://nodejs.org/)\n- **Git** - [Download](https://git-scm.com/)\n\n## 📥 Clone Repository\n\n```bash\ngit clone https://github.com/garrigaandres35-code/turf-analytics.git\ncd turf-analytics\n```\n\n## 🔙 Backend Setup (5 minutes)\n\n### 1. Create Virtual Environment\n\n```bash\ncd backend\npython -m venv venv\n\n# On Windows\n.\\venv\\Scripts\\activate\n\n# On macOS/Linux\nsource venv/bin/activate\n```\n\n### 2. Install Dependencies\n\n```bash\npip install -r requirements.txt\n```\n\n### 3. Setup Database\n\n```bash\npython manage.py migrate\npython manage.py createsuperuser  # Create admin user\n```\n\n### 4. Start Backend Server\n\n```bash\npython manage.py runserver 8000\n```\n\n✅ Backend running at: **http://localhost:8000**\n\n📚 API Docs: **http://localhost:8000/api/docs/**\n\n---\n\n## 🎨 Frontend Setup (5 minutes)\n\n### 1. Navigate to Frontend\n\n```bash\ncd ../frontend\n```\n\n### 2. Install Dependencies\n\n```bash\nnpm install\n```\n\n### 3. Setup Environment\n\nThe `.env` file already exists with default values for local development:\n\n```bash\n# .env is ready to use, but you can customize if needed\nREACT_APP_DOMINIO=localhost\nREACT_APP_PUERTO_BACKEND=8000\nREACT_APP_PUERTO_FRONTEND=3000\n```\n\n### 4. Start Frontend Server\n\n```bash\nnpm start\n```\n\n✅ Frontend running at: **http://localhost:3000**\n\n---\n\n## 🔐 Test Login\n\nOnce both servers are running, open http://localhost:3000 and login with:\n\n### Admin Account\n```\nEmail: admin@turf-analytics.com\nPassword: Admin123!@#\n```\n\n### Demo Account\n```\nEmail: demo@turf-analytics.com\nPassword: Demo123!@#\n```\n\n---\n\n## 📊 Test Features\n\n### Admin Panel\n1.
